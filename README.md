# ParentBud Web

React + TypeScript + Vite starter with TailwindCSS and a shadcn/ui-ready structure.

## Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- shadcn/ui-compatible aliases and helpers
- Framer Motion, Lucide React, and Radix Slot for open-source UI components

## Scripts

- `npm run dev` - start the dev server
- `npm run build` - typecheck and build for production
- `npm run preview` - preview the production build
- `npm run typecheck` - run TypeScript only

## Adding more UI

This scaffold is ready for components from:

- shadcn/ui
- Aceternity UI
- Uiverse
- React Bits

Use the aliases in `tsconfig.json` and `vite.config.ts` when adding components under `src/components` and `src/lib`.
